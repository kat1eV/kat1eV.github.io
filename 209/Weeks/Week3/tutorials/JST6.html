JavaScript Arrays

Why use arrays?

An array is a special variable, which can hold more than one value

It is especially helpful when trying to loop through a series of ojects to find the one you want

Array Syntax

const array_name = [item1, item2, ...];

can also use keyword new Array

Accessing Array Elements

you refer to index number

 i.e.
        const cars = ["Saab", "Volvo", "BMW"];
        let myCar = cars[0];

Changing an Array Element

        cars[0] = "Opel";

Converting an array to a string

        const fruits = ["Banana", "Orange", "Apple", "Mango"];
        document.getElementById("demo").innerHTML = fruits.toString();

        Result:
        Banana,Orange,Apple,Mango

Access Full Array

access by array name

        const cars = ["Saab", "Volvo", "BMW"];
        document.getElementById("demo").innerHTML = cars;

Arrays are Objects

but array elements can also be made objects:

        myArray[0] = Date.now;
        myArray[1] = myFunction;
        myArray[2] = myCars;
